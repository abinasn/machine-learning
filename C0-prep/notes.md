1. Sequences:- Predict what's next
  Example: 1, 2, 3, 4, 5, ?
  Answer: 6 -> +1
    
2. Odd one out:- Find what doesn't belong to the series
  Example: 2, 4, 6, 8, 10, 11
  Answer: 11 -> because all are even numbers except 11

3. Analogies:- Find and apply relationships
  Example: Pen is to Writer as Brush is to _____
  Answer: Painter

4. True or False:- Check if conclusion is true or false based on premises
  Example: 
  Facts:
    All dogs have 4 legs.
    Tommy is a dog.
  Conclusion: Tommy has 4 legs.
  Answer: True
